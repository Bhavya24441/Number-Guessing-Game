# Number-Guessing-Game
A simple Python game where the computer chooses a random number and the user tries to guess it.
## How it works
- The computer selects a random number between 1 and 100
- The user keeps guessing
- The program tells whether the guess is too high or too low
- The game ends when the correct number is guessed

## Concepts Used
- Loops
- Conditional statements
- Random module
- User input
- Program logic design

## How to run
```bash
python number_guessing_game.py
